# Instruções para Criação de Páginas de Artigos - Moreira & Ponciano Advogados

## Objetivo

Criar uma página HTML individual para **cada artigo** listado abaixo, seguindo como base/template a página `./advogados-familia/divorcio-direitos-que-a-mulher-tem.html` (página de um outro serviço).

## Regras Gerais

1. **Template base**: Use a página `./advogados-familia/divorcio-direitos-que-a-mulher-tem.html` como referência de layout, estrutura, estilo e componentes visuais.
2. **Menu ativo**: Em todas as páginas de artigos, o item **"Artigos Jurídicos"** no menu de navegação deve estar com a classe `active`.
3. **Conteúdo humanizado**: Cada página deve conter um texto **humanizado, interessante e persuasivo** sobre o respectivo serviço, explicando o assunto de forma acessível ao público leigo. O texto deve ser longo e detalhado, com foco em SEO.
4. **Importância do advogado**: Em cada página, explique a **importância de contar com um bom advogado** especializado no assunto.
5. **CTAs (Call to Action)**: Distribua **diversas CTAs ao longo da página** (botões de contato via WhatsApp etc.), não apenas no topo ou rodapé, sempre usar nosso whatsapp https://wa.me/5511944617258
6. **SEO e Meta Tags**: Cada página deve conter meta tags otimizadas e únicas:
   - `<title>` específico do artigo. Exemplos de metatags uteis:

    <meta property="og:type" content="article" />
